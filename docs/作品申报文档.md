# 《九价守护计划：宫颈癌预防科普大冒险》作品申报文档

## 一、作品简介

**作品名称：** 九价守护计划：宫颈癌预防科普大冒险

**作品类型：** H5 互动科普网页游戏

**作品主题：** 宫颈癌预防 · HPV 感染 · 九价 HPV 疫苗 · 接种流程 · 宫颈癌筛查 · 常见误区辨析

**适用场景：** 医学科普比赛、健康科普课程作业、公众健康教育展示

**目标受众：** 关注宫颈健康的公众、适龄人群、健康科普学习者

**作品说明：**
本作品是一款基于 HTML5 + CSS3 + JavaScript 开发的互动科普网页游戏。玩家扮演"宫颈健康守护员"，进入名为"健康花园"的科普城市，通过 6 个精心设计的互动任务，系统学习 HPV 病毒与宫颈癌的关系、九价 HPV 疫苗覆盖型别、疫苗预防机制、接种流程、筛查意识和谣言辨析。完成全部任务后获得"九价守护徽章"，并解锁结局评分和知识总结。

作品内置**演示模式**，适合在比赛或课堂中进行 4 分钟录屏演示，自动引导操作流程并提供口播提示文案。

## 二、代表性展示图说明

### 首页（健康花园入口）
卡通医疗风格背景，包含浮动的病毒粒子、宫颈健康守护盾动画。显示标题、副标题和免责声明。提供"开始游戏""演示模式""知识资料""参考来源"四个入口。

### 任务一：认识 HPV 病毒
2×N 网格排列的彩色卡片，包含 HPV 病毒、其他病毒、细菌、过敏原等选项。玩家点击正确 HPV 卡片后弹出科普卡片。

### 任务二：高/低危型别分拣站
拖拽+点击双模式交互。展示九价疫苗覆盖的 9 种 HPV 型别，玩家将型别分类到"低危相关型别区"和"高危相关型别区"。正确分类显示绿色光效。

### 任务三：免疫训练营
倒计时匹配游戏。左侧展示"抗原线索"（VLP 特征），右侧展示"免疫记忆细胞"，玩家需在限时内完成型别匹配，模拟疫苗训练免疫系统的过程。

### 任务四：接种流程模拟器
序列排序交互。玩家将 7 张接种相关流程卡片按正确顺序排列，完成排序后触发门诊盖章动画。

### 任务五：双重防线模拟
可视化盾牌合成动画。左侧"HPV 疫苗预防防线"和右侧"宫颈癌筛查防线"，玩家需同时点亮两道防线才能形成完整守护盾。

### 任务六：谣言粉碎机
6 道判断题，每题提供即时反馈和详细解释，粉碎关于 HPV 和九价疫苗的常见误区。

### 结局页
展示全部 6 枚徽章、宫颈健康守护指数、称号评定、知识总结入口。

## 三、科学原理及内容

### 3.1 HPV 与宫颈癌的关系
- HPV（人乳头瘤病毒）感染很常见，多数感染可被机体免疫系统清除
- 部分高危型 HPV（如 HPV16、18、31、33、45、52、58）持续感染可能增加宫颈癌及相关癌前病变风险
- 低危型 HPV（如 HPV6、11）常与生殖器疣相关

### 3.2 九价 HPV 疫苗
- 九价 HPV 疫苗覆盖 HPV 6、11、16、18、31、33、45、52、58 型
- 疫苗通过病毒样颗粒（VLP）呈现与病毒相关的特征，帮助免疫系统提前建立识别能力
- 疫苗不含活病毒，不能导致 HPV 感染
- 疫苗主要用于预防，不用于治疗已经发生的 HPV 感染或宫颈癌

### 3.3 宫颈癌预防策略
- 一级预防：HPV 疫苗接种
- 二级预防：宫颈癌筛查（如宫颈细胞学检查、HPV 检测等）
- 三级预防：发现异常后的及时随访和处理
- 接种 HPV 疫苗不能替代宫颈癌筛查，二者互补

### 3.4 接种注意事项
- 接种年龄、剂次、间隔、禁忌证和注意事项应以当地批准说明书和接种门诊建议为准
- 是否适合接种需咨询专业人员

## 四、创作目的

1. **提升公众认知：** 帮助公众了解 HPV 感染与宫颈癌的关系，提高宫颈癌预防意识
2. **传播正确信息：** 纠正关于九价 HPV 疫苗的常见误区
3. **强化预防观念：** 强调"疫苗+筛查"双重防线的重要性
4. **避免污名化：** 倡导科学、尊重的 HPV 科普方式
5. **创新科普形式：** 通过互动游戏提升科普传播的趣味性和参与度

## 五、设计思路

### 5.1 整体框架
采用"关卡制"游戏架构，将宫颈癌预防的 6 个核心知识点分别设计为 6 个互动任务。每个任务有明确的学习目标、交互方式和反馈机制。完成所有任务后进入结局页进行综合评分。

### 5.2 视觉风格
- 现代卡通医学科普风，温和、可信、明亮
- 避免恐怖化疾病表达，避免对 HPV 感染者污名化
- 主色调：医疗蓝 #2F80ED、健康绿 #27AE60、柔和粉 #FF8FB3、温暖黄 #F2C94C
- 背景浅蓝 #EEF7FF，卡片白 #FFFFFF

### 5.3 交互设计
- 多种交互方式：点击选择、拖拽分类、倒计时匹配、流程排序、防线点亮、真假判断
- 即时反馈：正确→绿色动画+加分，错误→温和提示+解释
- 徽章系统：完成任务获得对应徽章，激励玩家完成全部任务
- 演示模式：自动引导+高亮提示，适合录屏演示

### 5.4 医学严谨性
- 每个科普知识点均标注参考来源
- 多处加入科普提示和免责声明
- 谨慎使用"预防""降低风险"等措辞，避免"保证""一定"等绝对化表述
- 强调"以当地说明书和接种门诊建议为准"

## 六、交互流程

```
首页（健康花园入口）
  ├── 开始游戏 → 任务一（认识 HPV）→ 任务二（分拣站）→ 任务三（免疫训练营）
  │                 → 任务四（接种流程）→ 任务五（双重防线）→ 任务六（谣言粉碎）
  │                 → 结局页（评分+徽章+称号）
  ├── 演示模式 → 自动引导完成 6 个任务 → 结局页
  ├── 知识资料 → 6 个科普知识卡片
  └── 参考来源 → 5 个权威来源链接
```

每个任务流程：
```
任务说明 → 交互操作 → 即时反馈 → 完成任务 → 获得徽章 → 下一关 / 结局
```

## 七、技术实现说明

### 技术栈
- **HTML5：** 语义化结构、Section 屏幕切换
- **CSS3：** 自定义属性、Flexbox & Grid、CSS 动画、过渡效果、响应式设计
- **JavaScript (ES6+)：** 原生 JS，无框架依赖。游戏状态管理、事件驱动交互、计时器、动画控制

### 关键实现
- **屏幕管理：** 基于 CSS `display` 和 `active` 类的屏幕切换系统
- **游戏状态：** 集中式 state 对象管理所有游戏数据
- **演示模式：** 独立的演示标志 + 自动高亮 + 口播提示文案
- **响应式：** 移动优先设计，适配 320px~1024px+ 屏幕
- **离线运行：** 零外部依赖，打开 index.html 即可运行

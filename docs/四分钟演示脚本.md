# 《九价守护计划》四分钟演示脚本

## 录制前准备

1. 用 Chrome / Edge 打开 `index.html`
2. 按 F12 → 切换设备模拟 → 选择 iPhone 14 Pro (393×852) 或类似手机分辨率
3. 点击首页"**演示模式**"按钮
4. 按本脚本节奏录制 + 配音
